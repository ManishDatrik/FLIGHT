# Flight Controller 

## About the Project

This project is a Flight Controller PCB designed using EasyEDA. The aim of this project was to learn the complete PCB design process by creating a compact and organized hardware board for a flight controller.

I designed the schematic, selected the required electronic components, connected them correctly, converted the design into a PCB, routed the tracks, checked the design for errors, and generated the manufacturing files.

This project helped me understand how a real flight controller board is designed before it is manufactured.

---

## Purpose

A flight controller is the main board of a drone. It receives signals from different sensors and controls the motors to keep the drone stable during flight.

This PCB is designed to provide proper connections between different hardware components while keeping the board compact and easy to manufacture.

## What I Did

### 1. Created the Project

Started a new project in EasyEDA and created the schematic.

---

### 2. Designed the Schematic

Added all the required electronic components and connected them using proper nets.

The schematic includes power connections, communication lines, input/output connections, and supporting components.

---

### 3. Assigned Footprints

Assigned suitable footprints for every component before moving to PCB design.

---

### 4. Converted to PCB

Transferred the schematic to the PCB editor.

All the components were imported successfully for PCB placement.

---

### 5. Component Placement

Placed the components carefully to reduce routing complexity and make the PCB compact.

Special attention was given to connector placement and important signal paths.

---

### 6. PCB Routing

Connected all the components using PCB tracks.

The routing was done carefully to keep the board clean and reduce unnecessary crossings.

---

### 7. Design Check

Performed design checking to verify the PCB connections and correct any routing issues before generating manufacturing files.

---

### 8. Manufacturing Files

Generated all the required manufacturing files including:

- Gerber Files
- Drill Files
- PCB Manufacturing Data
